## Complexities

 ```sh
**queue**

enqueue()     = O(1)
dequeue()     = O(1)
first()       = O(1)
last()        = O(1)

**stack**

push()        = O(1)
top()         = O(1)
pop()         = O(1)

**search**

first_index() = O(n)
last_index()  = O(n)

**other**

display()     = O(n)
reverse()     = O(n)
```

 ```console
Elbouchouki:~$ git clone https://github.com/Elbouchouki/Algorithms.git
```

